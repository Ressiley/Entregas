Os principais protocolos de rede

Os protocolos de rede são conjuntos de normas que permitem que duas ou mais máquinas conectadas á internet se comuniquem entre si, funcionando com uma linguagem universal interpretada por computadores de qualquer fabricante, por meio de qualquer sistema operacional.

Protocolo TCP/IP

Definição

É um acrônomo de dois protocolos combinados, o TCP(Trasnmission Conntrol Protocol) ou Protocolo de Controle de Transmissão e IP (Internet Protocol) ou Protocolo de Internet.

Protocolo HTTP

Definição

O protocolo HTTP (Hypertext Transfer Protocol) ou Protocolo de Transferência de Hipertexto é um protocolo de transferência que possibilita que as pessoas que inserem a URL do seu site na Web possam ver os conteúdos e dados que nele existem.

Protocolo HTTPS

O que é

O protocolo HTTPS (Hypertext Transfer Protocol Secure) ou Protocolo de Transferência de Hipertexto Seguro é um protocolo de transferência que insere uma camada de proteção na transmissão de dados entre o seu computador e o servidor.
